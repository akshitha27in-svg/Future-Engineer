This folder is reserved for any images or media you want to add later
(e.g. a custom logo, favicon, or hero image).

The site currently works entirely with emoji icons and CSS-drawn graphics,
so no external image files are required for it to run.

To add a custom logo:
1. Drop your image file here, e.g. assets/logo.png
2. In index.html, replace the "FE" brand-mark spans with:
   <img src="assets/logo.png" alt="Future Engineer logo" style="width:38px;height:38px;border-radius:10px;">
